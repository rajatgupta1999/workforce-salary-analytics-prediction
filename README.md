# Workforce Salary Analytics & Prediction Platform

## Project Overview

This project analyzes workforce salary data to identify the key factors influencing employee compensation. Using Python, Machine Learning, and Power BI, the project explores salary trends across education, experience, company size, location, certifications, and remote work arrangements.

The project combines exploratory data analysis, predictive modelling, and business intelligence to generate actionable workforce insights.

---

## Dataset

* 250,000 employee records
* 10 workforce-related features
* Salary range: $31,867 – $333,046

Features include:

* Job Title
* Experience Years
* Education Level
* Skills Count
* Industry
* Company Size
* Location
* Remote Work
* Certifications
* Salary

---

## Key Findings

* Employees in the USA earned the highest salary premium (+$41.9K).
* Enterprise companies paid approximately $42K more than startups.
* PhD holders earned approximately $21.5K more than the baseline education group.
* Each additional year of experience increased salary by approximately $2.7K.
* Industry had minimal impact on salary compared to education, location, and company size.

---

## Machine Learning Results

| Model             | MAE   | R² Score |
| ----------------- | ----- | -------- |
| Linear Regression | 5,436 | 0.963    |
| Random Forest     | 5,693 | 0.961    |

Linear Regression outperformed Random Forest, indicating that salary relationships within the dataset are largely linear.
